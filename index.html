Password Generator

One of my early assignments was to create a password generator with different criteria selected by the user. First the user clicks on the start button and is prompted to set the password attributes. Parameters such as length, lowercase, uppercase, numeric, and/or special characters. After some brief questions are answered (with at least one-character type selected) the user is presented with a password written on the page.

Your Weather

The “Your Weather” site was the first time I encountered APIs. Using the “Open Weather Map” API to create a dashboard with a search form input. Upon searching the user is presented with current and future weather conditions for that city such as the temperature, the humidity, the wind speed, and the UV index. Plus, soon to be added a 5-day forecast that displays the date, an icon representation of weather conditions, the temperature, and the humidity.
